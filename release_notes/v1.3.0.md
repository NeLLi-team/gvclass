# GVClass v1.3.0

## Summary

- Added baseline-normalized order completeness scoring so complete isolates are not systematically penalized by low raw order-marker recovery.
- Preserved raw completeness values in the output alongside normalized completeness for transparent strategy comparison.
- Bumped the software version to `v1.3.0` because the summary output schema and completeness semantics changed.

## Details

- `order_completeness` is now normalized against the order-specific complete-reference baseline in `resources/order_completeness.tab`.
- Raw values are retained as `order_completeness_raw` and `weighted_order_completeness_raw`.
- Output now includes the order baseline mean, baseline standard deviation, reference order, and normalization strategy label.

## Notes

- The database version remains `v1.2.2`; this release changes software behavior, not the bundled database.
