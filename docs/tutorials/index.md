# Tutorials

Each lesson walks you through a complete task from start to finish, with a visible result at every step, so your first run succeeds. New to GVClass? Start with the lesson below. Later lessons build on it, so work through them in order.

- [Get started with GVClass](getting-started.md): install GVClass with pixi, set up the database bundle, and classify the bundled example end to end, from input genomes to a finished summary table.

!!! tip "Already know the basics?"
    If you have run GVClass before and want to solve a specific task, go to the [how-to guides](../how-to/index.md). To look up what a result means, use the [output reference](../reference/output.md). To understand the method behind the calls, read [how GVClass works](../explanation/how-it-works.md).
